# Website Field Guide

## Storm Channel categories

Suggested categories already supported:

- Full-Force Games
- Lightning Strikes
- Caught in the Storm
- In the Circle
- Behind the Storm
- Rising Storm
- Built by the Storm
- Control the Wind
- Own the Ground
- Protect the Eye

Paste either a YouTube video ID or a full YouTube URL in `YouTubeID`. The website extracts the ID automatically.

## Schedule status language

- Storm Forming — practice
- Storm Warning — game
- Storm Watch — tournament or tryout
- Storm Passed — completed event
- Next Impact — the next scheduled event

## Parent availability labels

- READY FOR THE STORM
- NOT AVAILABLE
- FORECAST UNCERTAIN

## Public approval rule

A row appears publicly only when both fields are set to `YES`:

- Show
- Approved


## Online media

- `YouTubeID`: YouTube URL or ID.
- `VideoURL`: Google Drive preview/share link or a clip uploaded from the Coach Edit Board.
- `ThumbnailURL`: optional custom thumbnail uploaded to Google Drive.
- Clips uploaded through the editor should be under 45 MB. Larger clips should be uploaded to Drive manually and linked.
